# Space Theme Asset Placeholders

This folder contains placeholder assets for the space refactor.

- `backgrounds/`: nebula, station, and deep-space backdrops.
- `player/`: astronaut concept placeholders.
- `traps/`: alien hazard concept placeholders (tentacle, drone, egg).
- `platforms/`: sci-fi deck, asteroid, and organic platform concepts.
- `ui/`: holographic panel and icon references.
- `sfx/`: sci-fi sound design notes and placeholders.

Current build uses procedural Phaser vector/canvas rendering for performance.
These files are ready targets for final art/sound replacements.
